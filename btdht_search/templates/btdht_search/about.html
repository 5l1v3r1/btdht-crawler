Overview

BTDHT Search is a BitTorrent DHT search engine.
BitTorrent is a file sharing protocol.
A DHT stand for Distributed Hash Table, the bittorent DHT, is a decentralized database used to find peers distributing content.
A torrent is a file with metadata which describes distributed content.

BTDHT Search's database is formed by DHT users.
BTDHT Search participates in the bittorent DHT network and follows the bittorent DHT protocol specification.
The project name is an acronym for Bittorrent Distributed Hash Table Search.

BTDHT Search is not a tracker and doesn't store any content and only collects torrent metadata (such as file names and file sizes) and a magnet link (torrent identifier).
